# activiteGit
Openclasrooms cours gérer son code avec Git et GitHub - activité 1

Cet exercice a pour objectif de mettre en pratique les notions acquises dans les deux premières parties du cours Openclassrooms sur Git et Github.

Ce repository Git contient :

- Un fichier nommé "README.txt" qui présente le projet;
- 2 autres fichiers (ils peuvent contenir des scripts de code, du texte sur un projet personnel...);
- 4 commits distincts montrant l’ajout puis la modification de certains fichiers.
